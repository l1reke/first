Основні команди Git

Ініціалізація

  • git init — створення нового репозиторію

Налаштування

  • git config --global user.name "Ім’я"
  • git config --global user.email "email@example.com"

Коміти

  • git add .
  • git commit -m "повідомлення"

Гілки

  • git branch — показати гілки
  • git checkout -b new-branch — створити та перейти до гілки

Злиття

  • git merge гілка — злиття гілки

Віддалений репозиторій

  • git remote add origin URL
  • git push -u origin main
